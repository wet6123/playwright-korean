## Introduction

Playwright는 대부분의 브라우저 기능에 대한 기본 지원을 제공합니다. 그러나 일부 실험적인 API와 모든 브라우저에서 (아직) 완전히 지원되지 않는 API가 있습니다. Playwright는 일반적으로 이러한 경우에 전용 자동화 API를 제공하지 않습니다. 이러한 경우에 모의를 사용하여 애플리케이션의 동작을 테스트할 수 있습니다. 이 가이드에서는 몇 가지 예제를 제공합니다.

## Introduction

장치의 배터리 상태를 표시하기 위해 [battery API](https://developer.mozilla.org/en-US/docs/Web/API/Navigator/getBattery)를 사용하는 웹 앱을 생각해 보겠습니다. 배터리 API를 모의하고 페이지가 배터리 상태를 올바르게 표시하는지 확인합니다.

## Creating mocks

페이지가 로드되는 동안 매우 초기에 API를 호출할 수 있으므로 페이지 로드가 시작되기 전에 모든 모의를 설정하는 것이 중요합니다. 이를 달성하는 가장 쉬운 방법은 [page.addInitScript()](/api/class-page.mdx#page-add-init-script)를 호출하는 것입니다.

```js
await page.addInitScript(() => {
  const mockBattery = {
    level: 0.75,
    charging: true,
    chargingTime: 1800,
    dischargingTime: Infinity,
    addEventListener: () => { }
  };
  // 모의 배터리 정보를 항상 반환하도록 메서드를 재정의합니다.
  window.navigator.getBattery = async () => mockBattery;
});
```

일단 이렇게 하면 페이지를 탐색하고 UI 상태를 확인할 수 있습니다.

```js
// 각 테스트 전에 모의 API를 구성합니다.
test.beforeEach(async ({ page }) => {
  await page.addInitScript(() => {
    const mockBattery = {
      level: 0.90,
      charging: true,
      chargingTime: 1800, // 초
      dischargingTime: Infinity,
      addEventListener: () => { }
    };
    // 모의 배터리 정보를 항상 반환하도록 메서드를 재정의합니다.
    window.navigator.getBattery = async () => mockBattery;
  });
});

test('show battery status', async ({ page }) => {
  await page.goto('/');
  await expect(page.locator('.battery-percentage')).toHaveText('90%');
  await expect(page.locator('.battery-status')).toHaveText('Adapter');
  await expect(page.locator('.battery-fully')).toHaveText('00:30');
});

```

## Mocking read-only APIs

일부 API는 읽기 전용이므로 navigator 속성에 할당할 수 없습니다. 예를 들어,

```js
// 다음 줄은 아무런 효과가 없습니다.
navigator.cookieEnabled = true;
```

그러나 속성이 [구성 가능](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object/defineProperty#configurable)인 경우 일반 JavaScript를 사용하여 재정의할 수 있습니다.

```js
await page.addInitScript(() => {
  Object.defineProperty(Object.getPrototypeOf(navigator), 'cookieEnabled', { value: false });
});
```

## Verifying API calls

경우에 따라 페이지가 예상되는 모든 API 호출을 수행했는지 확인하는 것이 유용합니다. 모든 API 메서드 호출을 기록한 다음 황금 결과와 비교할 수 있습니다. [page.exposeFunction()](/api/class-page.mdx#page-expose-function)는 페이지에서 테스트 코드로 메시지를 다시 전달하는 데 유용할 수 있습니다.

```js
test('log battery calls', async ({ page }) => {
  const log = [];
  // Node.js 스크립트로 메시지를 푸시하기 위한 함수를 노출합니다.
  await page.exposeFunction('logCall', msg => log.push(msg));
  await page.addInitScript(() => {
    const mockBattery = {
      level: 0.75,
      charging: true,
      chargingTime: 1800,
      dischargingTime: Infinity,
      // addEventListener 호출을 기록합니다.
      addEventListener: (name, cb) => logCall(`addEventListener:${name}`)
    };
    // 모의 배터리 정보를 항상 반환하도록 메서드를 재정의합니다.
    window.navigator.getBattery = async () => {
      logCall('getBattery');
      return mockBattery;
    };
  });

  await page.goto('/');
  await expect(page.locator('.battery-percentage')).toHaveText('75%');

  // 실제 호출을 황금 결과와 비교합니다.
  expect(log).toEqual([
    'getBattery',
    'addEventListener:chargingchange',
    'addEventListener:levelchange'
  ]);
});
```

## Updating mock

앱이 배터리 상태 업데이트를 올바르게 반영하는지 테스트하려면 모의 배터리 객체가 브라우저 구현과 동일한 이벤트를 발생시키는지 확인하는 것이 중요합니다. 다음 테스트는 이를 달성하는 방법을 보여줍니다.

```js
test('update battery status (no golden)', async ({ page }) => {
  await page.addInitScript(() => {
    // 배터리 상태가 변경될 때 해당 리스너에게 알리는 모의 클래스입니다.
    class BatteryMock {
      level = 0.10;
      charging = false;
      chargingTime = 1800;
      dischargingTime = Infinity;
      _chargingListeners = [];
      _levelListeners = [];
      addEventListener(eventName, listener) {
        if (eventName === 'chargingchange')
          this._chargingListeners.push(listener);
        if (eventName === 'levelchange')
          this._levelListeners.push(listener);
      }
      // 테스트에서 호출됩니다.
      _setLevel(value) {
        this.level = value;
        this._levelListeners.forEach(cb => cb());
      }
      _setCharging(value) {
        this.charging = value;
        this._chargingListeners.forEach(cb => cb());
      }
    }
    const mockBattery = new BatteryMock();
    // 모의 배터리 정보를 항상 반환하도록 메서드를 재정의합니다.
    window.navigator.getBattery = async () => mockBattery;
    // 더 쉽게 액세스할 수 있도록 모의 객체를 window에 저장합니다.
    window.mockBattery = mockBattery;
  });

  await page.goto('/');
  await expect(page.locator('.battery-percentage')).toHaveText('10%');

  // 레벨을 27.5%로 업데이트합니다.
  await page.evaluate(() => window.mockBattery._setLevel(0.275));
  await expect(page.locator('.battery-percentage')).toHaveText('27.5%');
  await expect(page.locator('.battery-status')).toHaveText('Battery');

  // 연결된 어댑터를 에뮬레이트합니다.
  await page.evaluate(() => window.mockBattery._setCharging(true));
  await expect(page.locator('.battery-status')).toHaveText('Adapter');
  await expect(page.locator('.battery-fully')).toHaveText('00:30');
});
```
