## Pages

각 [BrowserContext]는 여러 페이지를 가질 수 있습니다. [Page]는 브라우저 컨텍스트 내의 단일 탭 또는 팝업 창을 나타냅니다. URL로 이동하고 페이지 콘텐츠와 상호 작용하는 데 사용해야 합니다.

```js
// 페이지를 생성합니다.
const page = await context.newPage();

// 브라우저에서 URL을 입력하는 것과 유사하게 명시적으로 탐색합니다.
await page.goto('http://example.com');
// 입력을 채웁니다.
await page.locator('#search').fill('query');

// 링크를 클릭하여 암시적으로 탐색합니다.
await page.locator('#submit').click();
// 새 URL을 기대합니다.
console.log(page.url());
```

## Multiple pages

각 브라우저 컨텍스트는 여러 페이지 (탭)를 호스팅할 수 있습니다.
* 각 페이지는 포커스된 활성 페이지처럼 동작합니다. 페이지를 앞으로 가져올 필요가 없습니다.
* 컨텍스트 내부의 페이지는 뷰포트 크기, 사용자 정의 네트워크 경로 또는 브라우저 로케일과 같은 컨텍스트 수준 에뮬레이션을 따릅니다.

```js
// 두 개의 페이지를 만듭니다.
const pageOne = await context.newPage();
const pageTwo = await context.newPage();

// 브라우저 컨텍스트의 페이지를 가져옵니다.
const allPages = context.pages();
```

## Handling new pages

브라우저 컨텍스트의 `page` 이벤트를 사용하여 컨텍스트에서 생성된 새 페이지를 가져올 수 있습니다. 이는 `target="_blank"` 링크에 의해 열린 새 페이지를 처리하는 데 사용할 수 있습니다.

```js
// 클릭하기 전에 새 페이지를 기다리기 시작합니다. await는 없습니다.
const pagePromise = context.waitForEvent('page');
await page.getByText('open new tab').click();
const newPage = await pagePromise;
// 새 페이지와 정상적으로 상호 작용합니다.
await newPage.getByRole('button').click();
console.log(await newPage.title());
```

새 페이지를 트리거하는 작업을 알 수 없는 경우 다음 패턴을 사용할 수 있습니다.

```js
// 컨텍스트의 모든 새 페이지(팝업 포함)를 가져옵니다.
context.on('page', async page => {
  await page.waitForLoadState();
  console.log(await page.title());
});
```

## Handling popups

페이지가 팝업을 여는 경우 (`target="_blank"` 링크로 열린 페이지 등) 페이지에서 `popup` 이벤트를 수신하여 팝업에 대한 참조를 가져올 수 있습니다.

이 이벤트는 `browserContext.on('page')` 이벤트 외에도 발생하지만 이 페이지와 관련된 팝업에만 해당됩니다.

```js
// 클릭하기 전에 팝업을 기다리기 시작합니다. await는 없습니다.
const popupPromise = page.waitForEvent('popup');
await page.getByText('open the popup').click();
const popup = await popupPromise;
// 새 팝업과 정상적으로 상호 작용합니다.
await popup.getByRole('button').click();
console.log(await popup.title());
```

팝업을 트리거하는 작업을 알 수 없는 경우 다음 패턴을 사용할 수 있습니다.

```js
// 팝업이 열릴 때 모든 팝업을 가져옵니다.
page.on('popup', async popup => {
  await popup.waitForLoadState();
  console.log(await popup.title());
});
```
