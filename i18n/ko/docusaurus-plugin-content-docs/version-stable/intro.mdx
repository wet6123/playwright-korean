## Introduction

Playwright Test는 엔드 투 엔드 테스트의 요구 사항을 수용하기 위해 특별히 만들어졌습니다. Playwright는 Chromium, WebKit 및 Firefox를 포함한 모든 최신 렌더링 엔진을 지원합니다. Windows, Linux 및 macOS에서 로컬 또는 CI에서, 헤드리스 또는 헤드 모드로 Google Chrome for Android 및 Mobile Safari의 기본 모바일 에뮬레이션을 사용하여 테스트합니다.

## Installing Playwright

npm, yarn 또는 pnpm을 사용하여 Playwright를 설치하여 시작하십시오. 또는 [VS Code Extension](./getting-started-vscode.mdx)을 사용하여 테스트를 시작하고 실행할 수도 있습니다.

<Tabs defaultValue="npm" values={[ {label: 'npm', value: 'npm'}, {label: 'yarn', value: 'yarn'}, {label: 'pnpm', value: 'pnpm'} ] }>

<TabItem value="npm">

```bash
npm init playwright@latest
```

</TabItem>

<TabItem value="yarn">

```bash
yarn create playwright
```

</TabItem>

<TabItem value="pnpm">

```bash
pnpm create playwright
```

</TabItem>

</Tabs>

설치 명령을 실행하고 다음을 선택하여 시작하십시오.
- TypeScript 또는 JavaScript 중에서 선택하십시오 (기본값은 TypeScript).
- 테스트 폴더 이름 (기본값은 tests이거나 프로젝트에 이미 tests 폴더가 있는 경우 e2e).
- CI에서 테스트를 쉽게 실행할 수 있도록 GitHub Actions 워크플로우를 추가하십시오.
- Playwright 브라우저를 설치하십시오 (기본값은 true).

## What's Installed

Playwright는 필요한 브라우저를 다운로드하고 다음 파일을 만듭니다.

```bash
playwright.config.ts
package.json
package-lock.json
tests/
  example.spec.ts
tests-examples/
  demo-todo-app.spec.ts
```

[playwright.config](./test-configuration.mdx)는 Playwright를 실행할 브라우저 수정 등 Playwright에 대한 구성을 추가할 수 있는 곳입니다. 이미 존재하는 프로젝트 내에서 테스트를 실행하는 경우 종속성이 `package.json`에 직접 추가됩니다.

`tests` 폴더에는 테스트를 시작하는 데 도움이 되는 기본 예제 테스트가 포함되어 있습니다. 더 자세한 예제를 보려면 todo 앱을 테스트하기 위해 작성된 테스트가 포함된 `tests-examples` 폴더를 확인하십시오.

## Running the Example Test

기본적으로 테스트는 3개의 워커를 사용하여 Chromium, Firefox 및 WebKit의 3개 브라우저 모두에서 실행됩니다. 이는 [playwright.config 파일](./test-configuration.mdx)에서 구성할 수 있습니다. 테스트는 헤드리스 모드로 실행됩니다. 즉, 테스트를 실행할 때 브라우저가 열리지 않습니다. 테스트 결과 및 테스트 로그는 터미널에 표시됩니다.

## HTML Test Reports

테스트가 완료되면 [HTML Reporter](./test-reporters.mdx#html-reporter)가 생성됩니다. 이 보고서는 브라우저, 통과한 테스트, 실패한 테스트, 건너뛴 테스트 및 불안정한 테스트별로 보고서를 필터링할 수 있도록 테스트에 대한 전체 보고서를 보여줍니다. 각 테스트를 클릭하여 테스트의 오류와 테스트의 각 단계를 탐색할 수 있습니다. 기본적으로 일부 테스트가 실패하면 HTML 보고서가 자동으로 열립니다.

## Updating Playwright

Playwright를 최신 버전으로 업데이트하려면 다음 명령을 실행하십시오.

<Tabs defaultValue="npm" values={[ {label: 'npm', value: 'npm'}, {label: 'yarn', value: 'yarn'}, {label: 'pnpm', value: 'pnpm'} ] }>

<TabItem value="npm">

```bash
npm install -D @playwright/test@latest
# 또한 새 브라우저 바이너리 및 해당 종속성을 다운로드하십시오.
npx playwright install --with-deps
```

</TabItem>

<TabItem value="yarn">

```bash
yarn add --dev @playwright/test@latest
# 또한 새 브라우저 바이너리 및 해당 종속성을 다운로드하십시오.
yarn playwright install --with-deps
```

</TabItem>

<TabItem value="pnpm">

```bash
pnpm install --save-dev @playwright/test@latest
# 또한 새 브라우저 바이너리 및 해당 종속성을 다운로드하십시오.
pnpm exec playwright install --with-deps
```

</TabItem>

</Tabs>

다음 명령을 실행하여 Playwright의 버전을 항상 확인할 수 있습니다.

<Tabs defaultValue="npm" values={[ {label: 'npm', value: 'npm'}, {label: 'yarn', value: 'yarn'}, {label: 'pnpm', value: 'pnpm'} ] }>


</TabItem>

## System requirements
- Latest version of Node.js 18, 20 or 22.
- Windows 10+, Windows Server 2016+ or Windows Subsystem for Linux (WSL).
- macOS 13 Ventura, or later.
- Debian 12, Ubuntu 22.04, Ubuntu 24.04, on x86-64 and arm64 architecture.
