## Introduction

Playwright는 [`alert`](https://developer.mozilla.org/en-US/docs/Web/API/Window/alert), [`confirm`](https://developer.mozilla.org/en-US/docs/Web/API/Window/confirm), [`prompt`](https://developer.mozilla.org/en-US/docs/Web/API/Window/prompt) 및 [`beforeunload`](https://developer.mozilla.org/en-US/docs/Web/API/Window/beforeunload_event) 확인과 같은 웹 페이지 대화 상호 작용을 할 수 있습니다. 인쇄 대화 상자에 대해서는 [인쇄](#print-dialogs)를 참조하세요.

## alert(), confirm(), prompt() 대화 상자

기본적으로 대화 상자는 Playwright에 의해 자동으로 닫히므로 처리할 필요가 없습니다. 그러나 대화 상자를 트리거하는 액션 전에 대화 상자 핸들러를 등록하여 [dialog.accept()](/api/class-dialog.mdx#dialog-accept) 또는 [dialog.dismiss()](/api/class-dialog.mdx#dialog-dismiss)할 수 있습니다.

```js
page.on('dialog', dialog => dialog.accept());
await page.getByRole('button').click();
```

:::note
[page.on('dialog')](/api/class-page.mdx#page-event-dialog) 리스너는 반드시 대화 상자를 **처리해야 합니다**. 그렇지 않으면 [locator.click()](/api/class-locator.mdx#locator-click) 또는 다른 액션이 중단됩니다. 웹의 대화 상자는 모달이며 처리될 때까지 페이지 실행을 차단하기 때문입니다.
:::

결과적으로 다음 스니펫은 절대로 해결되지 않습니다.

:::warning

잘못되었습니다!
:::

```js
page.on('dialog', dialog => console.log(dialog.message()));
await page.getByRole('button').click(); // 여기서 멈춥니다.
```

:::note
[page.on('dialog')](/api/class-page.mdx#page-event-dialog)에 대한 리스너가 없으면 모든 대화 상자가 자동으로 닫힙니다.
:::

## beforeunload 대화 상자

[page.close()](/api/class-page.mdx#page-close)가 참인 [runBeforeUnload](/api/class-page.mdx#page-close-option-run-before-unload) 값으로 호출되면 페이지는 언로드 핸들러를 실행합니다. 이는 [page.close()](/api/class-page.mdx#page-close)가 페이지가 실제로 닫힐 때까지 기다리지 않는 유일한 경우입니다. 작업이 끝날 때 페이지가 열려 있을 수 있기 때문입니다.

다음에 나오는 것처럼 대화 상자 핸들러를 등록하여 `beforeunload` 대화 상자를 직접 처리할 수 있습니다.

```js
page.on('dialog', async dialog => {
  assert(dialog.type() === 'beforeunload');
  await dialog.dismiss();
});
await page.close({ runBeforeUnload: true });
```

## 인쇄 대화 상자

[`window.print`](https://developer.mozilla.org/en-US/docs/Web/API/Window/print)를 통해 인쇄 대화 상자가 트리거되었다고 주장하기 위해 다음 스니펫을 사용할 수 있습니다.

```js
await page.goto('<url>');

await page.evaluate('(() => {window.waitForPrintDialog = new Promise(f => window.print = f);})()');
await page.getByText('Print it!').click();

await page.waitForFunction('window.waitForPrintDialog');
```

이렇게 하면 버튼을 클릭한 후 인쇄 대화 상자가 열릴 때까지 기다립니다. 페이지가 로드된 후 / 버튼을 클릭하기 전에 스크립트를 평가해야 합니다.
