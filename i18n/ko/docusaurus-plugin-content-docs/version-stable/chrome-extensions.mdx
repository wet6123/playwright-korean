## 소개

:::note

확장 프로그램은 영구적인 컨텍스트로 실행된 Chrome/Chromium에서만 작동합니다. 사용자 지정 브라우저 인수는 Playwright 기능을 손상시킬 수 있으므로 사용 시 주의해야 합니다.
:::

아래 코드는 `./my-extension`에 있는 [Manifest v2](https://developer.chrome.com/docs/extensions/mv2/) 확장 프로그램의 [백그라운드 페이지](https://developer.chrome.com/extensions/background_pages)를 가져옵니다.

헤드리스 모드에서 확장 프로그램을 실행할 수 있도록 `chromium` 채널을 사용하는 것에 유의하세요. 또는 브라우저를 헤드 모드로 실행할 수도 있습니다.

```js
const { chromium } = require('playwright');

(async () => {
  const pathToExtension = require('path').join(__dirname, 'my-extension');
  const userDataDir = '/tmp/test-user-data-dir';
  const browserContext = await chromium.launchPersistentContext(userDataDir, {
    channel: 'chromium',
    args: [
      `--disable-extensions-except=${pathToExtension}`,
      `--load-extension=${pathToExtension}`,
    ],
  });
  let [backgroundPage] = browserContext.backgroundPages();
  if (!backgroundPage)
    backgroundPage = await browserContext.waitForEvent('backgroundpage');

  // 다른 페이지처럼 백그라운드 페이지를 테스트합니다.
  await browserContext.close();
})();
```

## 테스트

테스트를 실행할 때 확장 프로그램이 로드되도록 하려면 테스트 fixture를 사용하여 컨텍스트를 설정할 수 있습니다. 또한 확장 프로그램 ID를 동적으로 가져와서 팝업 페이지를 로드하고 테스트하는 데 사용할 수 있습니다.

헤드리스 모드에서 확장 프로그램을 실행할 수 있도록 `chromium` 채널을 사용하는 것에 유의하세요. 또는 브라우저를 헤드 모드로 실행할 수도 있습니다.

먼저, 확장 프로그램을 로드할 fixture를 추가합니다.

```js title="fixtures.ts"
import { test as base, chromium, type BrowserContext } from '@playwright/test';
import path from 'path';

export const test = base.extend<{
  context: BrowserContext;
  extensionId: string;
}>({
  context: async ({ }, use) => {
    const pathToExtension = path.join(__dirname, 'my-extension');
    const context = await chromium.launchPersistentContext('', {
      channel: 'chromium',
      args: [
        `--disable-extensions-except=${pathToExtension}`,
        `--load-extension=${pathToExtension}`,
      ],
    });
    await use(context);
    await context.close();
  },
  extensionId: async ({ context }, use) => {
    /*
    // manifest v2의 경우:
    let [background] = context.backgroundPages()
    if (!background)
      background = await context.waitForEvent('backgroundpage')
    */

    // manifest v3의 경우:
    let [background] = context.serviceWorkers();
    if (!background)
      background = await context.waitForEvent('serviceworker');

    const extensionId = background.url().split('/')[2];
    await use(extensionId);
  },
});
export const expect = test.expect;
```

그런 다음 테스트에서 이러한 fixture를 사용합니다.

```js
import { test, expect } from './fixtures';

test('예시 테스트', async ({ page }) => {
  await page.goto('https://example.com');
  await expect(page.locator('body')).toHaveText('Changed by my-extension');
});

test('팝업 페이지', async ({ page, extensionId }) => {
  await page.goto(`chrome-extension://${extensionId}/popup.html`);
  await expect(page.locator('body')).toHaveText('my-extension popup');
});
```
