Electron 애플리케이션 표현. [electron.launch()](/api/class-electron.mdx#electron-launch)를 사용하여 애플리케이션 인스턴스를 얻을 수 있습니다. 이 인스턴스를 사용하여 메인 Electron 프로세스를 제어하고 Electron 창을 사용할 수 있습니다.

```js
const { _electron: electron } = require('playwright');

(async () => {
  // Electron 앱을 실행합니다.
  const electronApp = await electron.launch({ args: ['main.js'] });

  // Electron 컨텍스트에서 표현식을 평가합니다.
  const appPath = await electronApp.evaluate(async ({ app }) => {
    // 이것은 메인 Electron 프로세스에서 실행되며, 여기서 매개변수는 항상
    // 메인 앱 스크립트에서 require('electron')의 결과입니다.
    return app.getAppPath();
  });
  console.log(appPath);

  // 앱이 여는 첫 번째 창을 가져오고, 필요한 경우 기다립니다.
  const window = await electronApp.firstWindow();
  // 제목을 출력합니다.
  console.log(await window.title());
  // 스크린샷을 캡처합니다.
  await window.screenshot({ path: 'intro.png' });
  // Electron 콘솔을 Node 터미널로 보냅니다.
  window.on('console', console.log);
  // 버튼을 클릭합니다.
  await window.click('text=Click me');
  // 앱을 종료합니다.
  await electronApp.close();
})();
```

### evaluate {#electron-application-evaluate}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font><x-search>electronApplication.evaluate</x-search>

[pageFunction](/api/class-electronapplication.mdx#electron-application-evaluate-option-expression)의 반환 값을 반환합니다.

[electronApplication.evaluate()](/api/class-electronapplication.mdx#electron-application-evaluate)에 전달된 함수가 [Promise]를 반환하면, [electronApplication.evaluate()](/api/class-electronapplication.mdx#electron-application-evaluate)는 Promise가 완료될 때까지 기다린 후 값을 반환합니다.

[electronApplication.evaluate()](/api/class-electronapplication.mdx#electron-application-evaluate)에 전달된 함수가 [Serializable]이 아닌 값을 반환하면, [electronApplication.evaluate()](/api/class-electronapplication.mdx#electron-application-evaluate)는 `undefined`를 반환합니다. Playwright는 또한 `JSON`으로 직렬화할 수 없는 일부 추가 값 (`-0`, `NaN`, `Infinity`, `-Infinity`)의 전송을 지원합니다.

**Usage**

```js
await electronApplication.evaluate(pageFunction);
await electronApplication.evaluate(pageFunction, arg);
```

**Arguments**
- `pageFunction` [function] | [Electron]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="electron-application-evaluate-option-expression"/><a href="#electron-application-evaluate-option-expression" class="list-anchor">#</a>
  
  메인 Electron 프로세스에서 평가할 함수입니다.
- `arg` [EvaluationArgument] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="electron-application-evaluate-option-arg"/><a href="#electron-application-evaluate-option-arg" class="list-anchor">#</a>
  
  [pageFunction](/api/class-electronapplication.mdx#electron-application-evaluate-option-expression)에 전달할 선택적 인수입니다.

**Returns**
- [Promise]&lt;[Serializable]&gt;<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="electron-application-evaluate-return"/><a href="#electron-application-evaluate-return" class="list-anchor">#</a>

### waitForEvent {#electron-application-wait-for-event}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.9</font><x-search>electronApplication.waitForEvent</x-search>

이벤트가 발생하기를 기다리고 해당 값을 predicate 함수에 전달합니다. predicate가 truthy 값을 반환하면 반환합니다. 이벤트가 발생하기 전에 애플리케이션이 닫히면 오류가 발생합니다. 이벤트 데이터 값을 반환합니다.

**Usage**

```js
const windowPromise = electronApp.waitForEvent('window');
await mainWindow.click('button');
const window = await windowPromise;
```

**Arguments**
- `event` [string]<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="electron-application-wait-for-event-option-event"/><a href="#electron-application-wait-for-event-option-event" class="list-anchor">#</a>
  
  이벤트 이름으로, 일반적으로 `*.on(event)`에 전달되는 것과 동일합니다.
- `optionsOrPredicate` [function] | [Object] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="electron-application-wait-for-event-option-options-or-predicate"/><a href="#electron-application-wait-for-event-option-options-or-predicate" class="list-anchor">#</a>
  - `predicate` [function]
    
    이벤트 데이터를 수신하고 대기해야 할 때 truthy 값으로 확인합니다.
  - `timeout` [number] *(optional)*
    
    대기할 최대 시간 (밀리초)입니다. 기본값은 `30000` (30초)입니다. 타임아웃을 비활성화하려면 `0`을 전달하십시오. 기본값은 [browserContext.setDefaultTimeout()](/api/class-browsercontext.mdx#browser-context-set-default-timeout)을 사용하여 변경할 수 있습니다.
  
  이벤트를 수신하는 predicate 또는 옵션 객체입니다. 선택 사항입니다.

**Returns**
- [Promise]&lt;[Object]&gt;<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="electron-application-wait-for-event-return"/><a href="#electron-application-wait-for-event-return" class="list-anchor">#</a>

### on('console') {#electron-application-event-console}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.42</font><x-search>electronApplication.on('console')</x-search>

Electron 메인 프로세스 내의 JavaScript가 `console.log` 또는 `console.dir`과 같은 콘솔 API 메서드 중 하나를 호출할 때 발생합니다.

`console.log`에 전달된 인수는 [ConsoleMessage] 이벤트 핸들러 인수에서 사용할 수 있습니다.

**Usage**

```js
electronApp.on('console', async msg => {
  const values = [];
  for (const arg of msg.args())
    values.push(await arg.jsonValue());
  console.log(...values);
});
await electronApp.evaluate(() => console.log('hello', 5, { foo: 'bar' }));
```

**Event data**
- [ConsoleMessage]
