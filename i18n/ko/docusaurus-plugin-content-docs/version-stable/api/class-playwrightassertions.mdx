Playwright는 예상 조건이 충족될 때까지 대기하고 재시도하는 어설션을 생성하는 편리한 메서드를 통해 웹 우선 어설션을 제공합니다.

다음 예제를 살펴보세요.

```js
import { test, expect } from '@playwright/test';

test('status becomes submitted', async ({ page }) => {
  // ...
  await page.locator('#submit-button').click();
  await expect(page.locator('.status')).toHaveText('Submitted');
});
```

Playwright는 가져온 Node에 `"Submitted"` 텍스트가 있을 때까지 선택자 `.status`로 노드를 다시 테스트합니다. 조건이 충족되거나 제한 시간에 도달할 때까지 노드를 반복적으로 다시 가져오고 확인합니다. 이 제한 시간은 옵션으로 전달할 수 있습니다.

기본적으로 어설션 제한 시간은 5초로 설정됩니다.
