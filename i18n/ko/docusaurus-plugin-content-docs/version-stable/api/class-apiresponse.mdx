### body {#api-response-body}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.16</font><x-search>apiResponse.body</x-search>

await apiResponse.body();

### dispose {#api-response-dispose}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.16</font><x-search>apiResponse.dispose</x-search>

이 응답의 본문을 폐기합니다. 호출되지 않으면 컨텍스트가 닫힐 때까지 본문이 메모리에 남아 있습니다.

**사용법**

```js
await apiResponse.dispose();
```

**반환값**
- [Promise]&lt;[void]&gt;<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="api-response-dispose-return"/><a href="#api-response-dispose-return" class="list-anchor">#</a>

### headers {#api-response-headers}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.16</font><x-search>apiResponse.headers</x-search>

이 응답과 관련된 모든 응답 HTTP 헤더가 포함된 객체입니다.

**사용법**

```js
apiResponse.headers();
```

**반환값**
- [Object]&lt;[string], [string]&gt;<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="api-response-headers-return"/><a href="#api-response-headers-return" class="list-anchor">#</a>

### headersArray {#api-response-headers-array}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.16</font><x-search>apiResponse.headersArray</x-search>

이 응답과 관련된 모든 응답 HTTP 헤더가 포함된 배열입니다. 헤더 이름은 소문자로 변환되지 않습니다. `Set-Cookie`와 같이 여러 항목이 있는 헤더는 배열에 여러 번 나타납니다.

**사용법**

```js
apiResponse.headersArray();
```

**반환값**
- [Array]&lt;[Object]&gt;<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="api-response-headers-array-return"/><a href="#api-response-headers-array-return" class="list-anchor">#</a>
  - `name` [string]

    헤더의 이름입니다.
  - `value` [string]

    헤더의 값입니다.

### json {#api-response-json}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.16</font><x-search>apiResponse.json</x-search>

응답 본문의 JSON 표현을 반환합니다.

이 메서드는 응답 본문을 `JSON.parse`로 구문 분석할 수 없는 경우 예외를 발생시킵니다.

**사용법**

```js
await apiResponse.json();
```

**반환값**
- [Promise]&lt;[Serializable]&gt;<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="api-response-json-return"/><a href="#api-response-json-return" class="list-anchor">#</a>
