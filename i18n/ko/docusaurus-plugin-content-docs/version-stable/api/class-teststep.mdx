## Properties

### annotations {#test-step-annotations}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.51</font><x-search>testStep.annotations</x-search>

현재 테스트 단계에 적용 가능한 어노테이션 목록입니다.

**Usage**

```js
testStep.annotations
```

**Type**
- [Array]&lt;[Object]&gt;
  - `type` [string]
    
    어노테이션 유형, 예를 들어 `'skip'`입니다.
  - `description` [string] *(optional)*
    
    선택적 설명입니다.

---

### attachments {#test-step-attachments}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.50</font><x-search>testStep.attachments</x-search>

[testInfo.attach()](/api/class-testinfo.mdx#test-info-attach)를 통해 단계 실행에 첨부된 파일 또는 버퍼 목록입니다.

**Usage**

```js
testStep.attachments
```

**Type**
- [Array]&lt;[Object]&gt;
  - `name` [string]
    
    첨부 파일 이름입니다.
  - `contentType` [string]
    
    리포트에서 적절하게 표시하기 위한 이 첨부 파일의 콘텐츠 유형입니다(예: `'application/json'` 또는 `'image/png'`).
  - `path` [string] *(optional)*
    
    첨부된 파일에 대한 파일 시스템의 선택적 경로입니다.
  - `body` [Buffer] *(optional)*
    
    파일 대신 사용되는 선택적 첨부 파일 본문입니다.

### category {#test-step-category}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.10</font><x-search>testStep.category</x-search>

다양한 출처와 상세도를 가진 단계를 구별하기 위한 단계 카테고리입니다. 내장된 카테고리는 다음과 같습니다.
* fixtures 및 hooks 초기화 및 해체를 위한 `hook`
* expect 호출을 위한 `expect`
* Playwright API 호출을 위한 `pw:api`.
* test.step API 호출을 위한 `test.step`.

**Usage**

```js
testStep.category
```

**Type**
- [string]
