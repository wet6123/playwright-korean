[ConsoleMessage] 객체는 [page.on('console')](/api/class-page.mdx#page-event-console) 이벤트를 통해 페이지에서 전달됩니다. 페이지에 기록된 각 콘솔 메시지에 대해 Playwright 컨텍스트에 해당 이벤트가 발생합니다.

```js
// 모든 콘솔 로그 수신
page.on('console', msg => console.log(msg.text()));

// 모든 콘솔 이벤트를 수신하고 오류 처리
page.on('console', msg => {
  if (msg.type() === 'error')
    console.log(`오류 텍스트: "${msg.text()}"`);
});

// 다음 콘솔 로그 가져오기
const msgPromise = page.waitForEvent('console');
await page.evaluate(() => {
  console.log('hello', 42, { foo: 'bar' });  // 페이지 내에서 console.log 실행
});
const msg = await msgPromise;

// 콘솔 로그 인수 분해
await msg.args()[0].jsonValue(); // hello
await msg.args()[1].jsonValue(); // 42
```
