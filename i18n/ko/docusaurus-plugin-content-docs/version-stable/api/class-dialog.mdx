[Dialog] 객체는 [page.on('dialog')](/api/class-page.mdx#page-event-dialog) 이벤트를 통해 페이지에서 전달됩니다.

`Dialog` 클래스 사용 예시:

```js
const { chromium } = require('playwright');  // 또는 'firefox' 또는 'webkit'.

(async () => {
  const browser = await chromium.launch();
  const page = await browser.newPage();
  page.on('dialog', async dialog => {
    console.log(dialog.message()); // 대화상자 메시지를 출력합니다.
    await dialog.dismiss(); // 대화상자를 닫습니다.
  });
  await page.evaluate(() => alert('1')); // '1'이라는 메시지를 가진 경고창을 띄웁니다.
  await browser.close(); // 브라우저를 닫습니다.
})();
```

:::note
[page.on('dialog')](/api/class-page.mdx#page-event-dialog) 리스너가 없는 한 대화상자는 자동으로 닫힙니다. 리스너가 있는 경우, 리스너는 반드시 [dialog.accept()](/api/class-dialog.mdx#dialog-accept) 또는 [dialog.dismiss()](/api/class-dialog.mdx#dialog-dismiss)를 호출하여 대화상자를 처리해야 합니다. 그렇지 않으면 페이지는 대화상자를 기다리며 [멈추게 되며](https://developer.mozilla.org/en-US/docs/Web/JavaScript/EventLoop#never_blocking), 클릭과 같은 작업이 완료되지 않습니다.
:::

## Methods

### accept {#dialog-accept}

<font size="2" style={{position: "relative", top: "-20px"}}>Added before v1.9</font><x-search>dialog.accept</x-search>

대화상자가 수락되었을 때 반환됩니다.

**사용법**

```js
await dialog.accept();
await dialog.accept(promptText);
```

**인수**
- `promptText` [string] *(선택사항)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="dialog-accept-option-prompt-text"/><a href="#dialog-accept-option-prompt-text" class="list-anchor">#</a>

  프롬프트에 입력할 텍스트입니다. 대화상자의 `type`이 프롬프트가 아니면 아무런 효과가 없습니다. 선택사항입니다.

**반환값**
- [Promise]&lt;[void]&gt;<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="dialog-accept-return"/><a href="#dialog-accept-return" class="list-anchor">#</a>

### defaultValue {#dialog-default-value}

<font size="2" style={{position: "relative", top: "-20px"}}>Added before v1.9</font><x-search>dialog.defaultValue</x-search>

**사용법**

```js
dialog.defaultValue();
```
