`CDPSession` 인스턴스는 원시 Chrome Devtools 프로토콜과 통신하는 데 사용됩니다:
* 프로토콜 메서드는 `session.send` 메서드로 호출할 수 있습니다.
* 프로토콜 이벤트는 `session.on` 메서드로 구독할 수 있습니다.

유용한 링크:
* DevTools 프로토콜 문서는 여기에서 찾을 수 있습니다: [DevTools Protocol Viewer](https://chromedevtools.github.io/devtools-protocol/).
* DevTools 프로토콜 시작하기: https://github.com/aslushnikov/getting-started-with-cdp/blob/master/README.md

```js
const client = await page.context().newCDPSession(page);
await client.send('Animation.enable');
client.on('Animation.animationCreated', () => console.log('애니메이션 생성됨!'));
const response = await client.send('Animation.getPlaybackRate');
console.log('재생 속도는 ' + response.playbackRate);
await client.send('Animation.setPlaybackRate', {
  playbackRate: response.playbackRate / 2
});
```

### detach {#cdp-session-detach}

<font size="2" style={{position: "relative", top: "-20px"}}>v1.9 이전에 추가됨</font><x-search>cdpSession.detach</x-search>

CDPSession을 대상에서 분리합니다. 일단 분리되면 CDPSession 객체는 이벤트를 발생시키지 않으며 메시지를 보내는 데 사용할 수 없습니다.

**사용법**

```js
await cdpSession.detach();
```
