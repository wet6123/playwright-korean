Coverage는 페이지에서 사용된 JavaScript 및 CSS 부분에 대한 정보를 수집합니다.

페이지 로드에 대한 Istanbul 보고서를 생성하기 위해 JavaScript coverage를 사용하는 예시:

:::note

Coverage API는 Chromium 기반 브라우저에서만 지원됩니다.
:::

```js
const { chromium } = require('playwright');
const v8toIstanbul = require('v8-to-istanbul');

(async () => {
  const browser = await chromium.launch();
  const page = await browser.newPage();
  await page.coverage.startJSCoverage();
  await page.goto('https://chromium.org');
  const coverage = await page.coverage.stopJSCoverage();
  for (const entry of coverage) {
    const converter = v8toIstanbul('', 0, { source: entry.source });
    await converter.load();
    converter.applyCoverage(entry.functions);
    console.log(JSON.stringify(converter.toIstanbul()));
  }
  await browser.close();
})();
```

### startJSCoverage {#coverage-start-js-coverage}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.11</font><x-search>coverage.startJSCoverage</x-search>

coverage가 시작되었는지 반환합니다

:::note

익명 스크립트는 연결된 URL이 없는 스크립트입니다. 이러한 스크립트는 `eval` 또는 `new Function`을 사용하여 페이지에서 동적으로 생성된 스크립트입니다. [reportAnonymousScripts](/api/class-coverage.mdx#coverage-start-js-coverage-option-report-anonymous-scripts)가 `true`로 설정된 경우 익명 스크립트의 URL은 `__playwright_evaluation_script__`가 됩니다.
:::

**Usage**

```js
await coverage.startJSCoverage();
await coverage.startJSCoverage(options);
```

**Arguments**
- `options` [Object] *(optional)*
  - `reportAnonymousScripts` [boolean] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="coverage-start-js-coverage-option-report-anonymous-scripts"/><a href="#coverage-start-js-coverage-option-report-anonymous-scripts" class="list-anchor">#</a>
    
    페이지에서 생성된 익명 스크립트를 보고할지 여부입니다. 기본값은 `false`입니다.
  - `resetOnNavigation` [boolean] *(optional)*<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="coverage-start-js-coverage-option-reset-on-navigation"/><a href="#coverage-start-js-coverage-option-reset-on-navigation" class="list-anchor">#</a>
    
    탐색할 때마다 coverage를 재설정할지 여부입니다. 기본값은 `true`입니다.

**Returns**
- [Promise]&lt;[void]&gt;<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="coverage-start-js-coverage-return"/><a href="#coverage-start-js-coverage-return" class="list-anchor">#</a>

### stopCSSCoverage {#coverage-stop-css-coverage}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.11</font><x-search>coverage.stopCSSCoverage</x-search>

모든 스타일시트에 대한 coverage 보고서 배열을 반환합니다.

:::note

CSS Coverage는 sourceURL이 없는 동적으로 삽입된 스타일 태그는 포함하지 않습니다.
:::

**Usage**

```js
await coverage.stopCSSCoverage();
```

**Returns**
- [Promise]&lt;[Array]&lt;[Object]&gt;&gt;<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="coverage-stop-css-coverage-return"/><a href="#coverage-stop-css-coverage-return" class="list-anchor">#</a>
  - `url` [string]
    
    StyleSheet URL
  - `text` [string] *(optional)*
    
    사용 가능한 경우 StyleSheet 콘텐츠
  - `ranges` [Array]&lt;[Object]&gt;
    - `start` [number]
      
      text의 시작 오프셋 (포함)
    - `end` [number]
      
      text의 끝 오프셋 (제외)
    
    사용된 StyleSheet 범위입니다. 범위는 정렬되고 겹치지 않습니다.

### stopJSCoverage {#coverage-stop-js-coverage}

<font size="2" style={{position: "relative", top: "-20px"}}>Added in: v1.11</font><x-search>coverage.stopJSCoverage</x-search>

모든 스크립트에 대한 coverage 보고서 배열을 반환합니다.

:::note

JavaScript Coverage는 기본적으로 익명 스크립트를 포함하지 않습니다. 그러나 sourceURL이 있는 스크립트는 보고됩니다.
:::

**Usage**

```js
await coverage.stopJSCoverage();
```

**Returns**
- [Promise]&lt;[Array]&lt;[Object]&gt;&gt;<a aria-hidden="true" tabIndex="-1" class="list-anchor-link" id="coverage-stop-js-coverage-return"/><a href="#coverage-stop-js-coverage-return" class="list-anchor">#</a>
  - `url` [string]
    
    Script URL
  - `scriptId` [string]
    
    Script ID
  - `source` [string] *(optional)*
    
    적용 가능한 경우 스크립트 콘텐츠
  - `functions` [Array]&lt;[Object]&gt;
    - `functionName` [string]
      
      
    - `isBlockCoverage` [boolean]
      
      
    - `ranges` [Array]&lt;[Object]&gt;
      - `count` [number]
        
        
      - `startOffset` [number]
        
        
      - `endOffset` [number]
        
        
      
    V8 특정 coverage 형식.
