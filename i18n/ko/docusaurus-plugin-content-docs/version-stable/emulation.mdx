## Introduction

Playwright를 사용하면 모든 브라우저에서 앱을 테스트하고 휴대폰이나 태블릿과 같은 실제 장치를 에뮬레이션할 수 있습니다. 에뮬레이션하려는 장치를 구성하기만 하면 Playwright는 `"userAgent"`, `"screenSize"`, `"viewport"`와 같은 브라우저 동작과 `"hasTouch"` 활성화 여부를 시뮬레이션합니다. 또한 모든 테스트 또는 특정 테스트에 대해 `"geolocation"`, `"locale"` 및 `"timezone"`을 에뮬레이션하고 알림을 표시하거나 `"colorScheme"`을 변경하기 위해 `"permissions"`를 설정할 수도 있습니다.

## Devices

Playwright는 선택한 데스크톱, 태블릿 및 모바일 장치에 대해 [playwright.devices](/api/class-playwright.mdx#playwright-devices)를 사용하여 [장치 매개변수 레지스트리](https://github.com/microsoft/playwright/blob/main/packages/playwright-core/src/server/deviceDescriptorsSource.json)와 함께 제공됩니다. 사용자 에이전트, 화면 크기, 뷰포트 및 터치 활성화 여부와 같은 특정 장치에 대한 브라우저 동작을 시뮬레이션하는 데 사용할 수 있습니다. 모든 테스트는 지정된 장치 매개변수로 실행됩니다.

## Viewport

뷰포트는 장치에 포함되어 있지만 [page.setViewportSize()](/api/class-page.mdx#page-set-viewport-size)를 사용하여 일부 테스트에 대해 재정의할 수 있습니다.

<Tabs
  groupId="js-flavor"
  defaultValue="test"
  values={[
    {label: 'Test', value: 'test'},
    {label: 'Library', value: 'library'}
  ]
}>
<TabItem value="test">

```js title="playwright.config.ts"
import { defineConfig, devices } from '@playwright/test';

export default defineConfig({
  projects: [
    {
      name: 'chromium',
      use: {
        ...devices['Desktop Chrome'],
        // `devices`를 구조 분해한 후에 `viewport` 속성을 정의하는 것이 중요합니다.
        // `devices`가 해당 장치에 대한 `viewport`도 정의하기 때문입니다.
        viewport: { width: 1280, height: 720 },
      },
    },
  ]
});
```

## isMobile

메타 뷰포트 태그가 고려되고 터치 이벤트가 활성화되었는지 여부입니다.

```js title="playwright.config.ts"
import { defineConfig, devices } from '@playwright/test';

export default defineConfig({
  projects: [
    {
      name: 'chromium',
      use: {
        ...devices['Desktop Chrome'],
        // `devices`를 구조 분해한 후에 `isMobile` 속성을 정의하는 것이 중요합니다.
        // `devices`가 해당 장치에 대한 `isMobile`도 정의하기 때문입니다.
        isMobile: false,
      },
    },
  ]
});
```

## Locale & Timezone

사용자 로케일 및 시간대를 에뮬레이션합니다. 구성에서 모든 테스트에 대해 전역적으로 설정한 다음 특정 테스트에 대해 재정의할 수 있습니다.

```js title="playwright.config.ts"
import { defineConfig } from '@playwright/test';

export default defineConfig({
  use: {
    // 사용자 로케일 에뮬레이션.
    locale: 'en-GB',

    // 사용자 시간대 에뮬레이션.
    timezoneId: 'Europe/Paris',
  },
});
```

## Geolocation

`"geolocation"` 권한을 부여하고 지리적 위치를 특정 영역으로 설정합니다.

```js title="playwright.config.ts"
import { defineConfig } from '@playwright/test';

export default defineConfig({
  use: {
    // 컨텍스트 지리적 위치
    geolocation: { longitude: 12.492507, latitude: 41.889938 },
    permissions: ['geolocation'],
  },
});
```

## Color Scheme and Media

사용자의 `"colorScheme"`을 에뮬레이션합니다. 지원되는 값은 'light' 및 'dark'입니다. [page.emulateMedia()](/api/class-page.mdx#page-emulate-media)를 사용하여 미디어 유형을 에뮬레이션할 수도 있습니다.

```js title="playwright.config.ts"
import { defineConfig } from '@playwright/test';

export default defineConfig({
  use: {
    colorScheme: 'dark',
  },
});
```

  groupId="js-flavor"

  values={[
    {label: 'Test', value: 'test'},
    {label: 'Library', value: 'library'}
  ]

<TabItem value="test">

```js title="tests/example.spec.ts"
import { test, expect } from '@playwright/test';

test.use({
  colorScheme: 'dark' // or 'light'
});

test('my test with dark mode', async ({ page }) => {
  // ...
});
```

</TabItem>
<TabItem value="library">

```js
// 다크 모드로 컨텍스트 생성
const context = await browser.newContext({
  colorScheme: 'dark' // or 'light'
});

// 다크 모드로 페이지 생성
const page = await browser.newPage({
  colorScheme: 'dark' // or 'light'
});

// 페이지의 색 구성표 변경
await page.emulateMedia({ colorScheme: 'dark' });

// 페이지의 미디어 변경
await page.emulateMedia({ media: 'print' });
```

</TabItem>
